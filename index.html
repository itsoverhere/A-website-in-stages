Whibble
An app I supervised with student-developers Victoria Angela Acorda, Richelle Chua, and Jed Pangilinan. Having trouble picking out a fun game to play with your friends? Whibble is a game where laughter is guaranteed. It is based on the telephone/whisper game but tweaked with drawings and guesses! At the end of the game, everyone gets to see how each player’s drawings and guesses can turn a word into something funny! No drawing skills required! Just have fun and enjoy the unexpected evolutions of the words! Whibble. Scribble the guess. Guess the scribble.
Loot
An app I supervised with student-developers Chino Tapales, Baldip Bhuller, and Joseph Manto. Loot is not your ordinary expense tracker. Aside from keeping track of your expenses and savings, Loot can also reward you for your actions. Think of it more like a game, the more you play, the better you get! In this case, the more money you save! Loot is dead simple to use with its clean and simple interface. With Loot, you can: - See exactly your money goes - Effortlessly add expenses - Easily manage your expenses - Securely view your expenses - Track your account and unlock achievements
Copy Simon
An app I supervised with student-developers Raafi Bandrang, Allen Gapuz, JM Hilario. How good do you think your memory is? How about your reflexes? Put it to the test in the game inspired by the classic Simon from our childhood game comes the application Copy Simon: The Ultimate Memory Game. Copy Simon is a simple yet addictive memory game. The game itself consists of 4 buttons in the middle which by default are green, yellow, red and, blue arranged in a 2 x 2 format. When the game starts, the buttons would light up in a random order and play sounds that vary per button. The player then would have to repeat the same sequence depending on the color and order that it lit up. When the player inputs the wrong sequence then it’s GAME OVER. If the user’s score is high enough then it prompts the user to input his name to be saved in the top 10 local high scores list on its respective game mode.
iTicket
An app developed by Neil Capistrano, Julia Lopez-Pozas, and Glenn Reyes, with the my supervision. iTicket makes use of NFC to validate tickets from attendee to event organizer. In doing so, it reduces the need for physical tickets and lining up to receive them. This app can be used for concerts, seminars, plays, parties, and all events in which you wish to manage attendance efficiently! Create, manage events. Reserve, validate tickets. All in one app.
Guess the Food
An app developed by Joanna Lau, Miguel Gomez, Ramon Panganiban with the my supervision. Guess The Food is a mobile game application that tests users of their knowledge on food names. The application has multiple levels in which the user is given a set of cartoon images or icons of food. The game will have multiple food packages such as food & vegetables, junk food, etc. A coin system is present in the game and will aid the user in guessing by using the coins to buy hints. Coins will be given based on how fast the user guesses the name of the food.
